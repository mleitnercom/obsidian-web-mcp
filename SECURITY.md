# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Use GitHub Private Vulnerability Reporting for this repository to disclose vulnerabilities privately.

Include:

- a short description of the issue
- affected configuration, environment, or version
- reproduction steps or a proof of concept, if available
- expected impact

## Supported Versions

Security fixes are provided for the current `main` branch and the latest release line.
